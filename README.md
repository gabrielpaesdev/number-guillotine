# Number Guillotine

Number Guillotine is a number guessing game written in C using GTK+3.  
The player must discover a secret number between 1 and 100 within a limited number of attempts.

---

## How to Play

1. Choose a difficulty:
   - Easy: 8 attempts  
   - Medium: 6 attempts  
   - Hard: 4 attempts  

2. Click a number between 1 and 100.
3. The game will tell you if the secret number is:
   - Higher  
   - Lower  

4. Numbers outside the valid range are crossed out and disabled.
5. You win if you guess the correct number before running out of attempts.

---

## Requirements

- GCC compiler  
- GTK+3 development libraries  
- Linux or any system with GTK+3 support
